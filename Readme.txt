Projeto simples ultilizando um sensor de temperatura modelo DHT11, junto com um display LCD 16x2, ultilizando as bibliotecas DHT, Adafruit_Sensor-master e LiquidCrystal.
----------------------------------------------------
Componetes 

1 Arduino Uno
1 Sensor de temperatura DHT11
1 LCD 16x2
1 Protoboard 830 furos.
1 Potenciometro 10K
